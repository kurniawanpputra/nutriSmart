==PROJECT DETAILS
* Ruby version: 2.3.4
* Rails version: 5.1.6
* DBMS: pgsql

==STEPS TO RUN THE PROJECT
1. Download the project
2. Make sure Rails is installed on your machine
3. Navigate to the project folder
4. Run in Git Bash command line "rails s"
5. Open the project in browser at localhost:3000
6. To shutdown the server, Ctrl+C in the command line
